## :pencil2: Overview

I created a blog with next.js. The article is generated by a static generator.

## :hammer: Requirement

- macOS
- Next.js 10.0.6
- [Material UI](https://material-ui.com/)

## :pushpin: Usage

Git clone
```
$ git clone https://github.com/shumatsumoto/next-blog.git
```
Move to directory
```
$ cd next-blog
```
Install the library
```
$ npm install
```
Launch a local server
```
$ npm run dev
```
Access URL： 
http://localhost:3000

## :railway_car: Features

- Article posting
- Tab Sort
- Pagination
